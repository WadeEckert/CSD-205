# CSD 205 – Introduction to Python
Wade Eckert's repository for CSD 205 - Introduction to Python.

This repository contains my coursework for **CSD 205 – Introduction to Python** at **Bellevue University**.

This course introduced the fundamentals of computer programming using the Python programming language. Throughout the course, I developed a strong foundation in problem solving, algorithm design, object-oriented programming, and core programming concepts that prepared me for more advanced software development coursework.

## Repository Structure

The repository is organized by module, with each folder containing the assignments, source code, and supporting documentation completed during that portion of the course.

```
module-1
module-2
...
module-11
```

Each module represents a different programming topic that builds upon concepts introduced in previous assignments.

## Topics Covered

Throughout this course, I gained experience with many of the core concepts of Python programming, including:

- Variables and data types
- User input and output
- Mathematical calculations
- Decision structures (if/else)
- Loops and iteration
- Functions
- File input and output
- Tuples
- Dictionaries
- String manipulation
- Classes and objects
- Object-oriented programming (OOP)
- Constructors and methods
- Inheritance
- Recursive and non-recursive algorithms
- Problem solving and algorithm design

## Major Programming Assignments

Projects completed throughout the course include:

- Fiber-optic installation cost calculator

- Tuple creation and iteration exercises
